# Mail Signature Generator

Mail Signature Generator, e-posta imzası oluşturmak için hazırlanmış basit bir web aracıdır.

İsim, unvan, şirket, iletişim bilgileri, sosyal medya bağlantıları, renkler, fotoğraf ve imza stilini düzenleyerek kendine uygun bir imza tasarlayabilirsin. Oluşturduğun imzayı kopyalayıp Gmail veya benzeri e-posta servislerindeki imza alanına yapıştırabilirsin.

Uygulama herhangi bir hesap gerektirmez. Girdiğin bilgiler kaydedilmez; imzanı sadece tarayıcında oluşturur ve kopyalamanı sağlar.
